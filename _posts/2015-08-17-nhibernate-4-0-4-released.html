NHibernate 4.0.4, 3.4.1, and 3.3.5 are now available for download from Sourceforge (4.0.4, 3.4.1, 3.3.5) and Nuget (4.0.4, 3.4.1, 3.3.5).

These releases fix regression caused by new C# Compiller "Roslyn". We are highly recommend to update if you are using or planing to use Visual Studio 2015 with NHibernate.

Bug

  • [NH-3795] - C# compiler "Roslyn" regression
